En el repositorio se subió el dockerfile, los programas a ejecutar y el archivo a leer.

Lo que se debe hacer es clonar el repositorio (git clone https://github.com/ppistarelli/tuia directorio-destino) en el equipo.
Luego ingresar al directorio y hacer el build (sudo docker build .)
Se ejecutará el archivo menu.sh que muestra el menú con los distintos programas a utilizar.

En el menu.sh se leen los archivos dentro del directorio programas y se muestran como puntos del menú. A medida que se agregan programas en el directorio aparecen automáticamente en el menú al ejecutarlo.
